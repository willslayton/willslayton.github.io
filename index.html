projects


Over the past few years I've worked on many projects so here's a list of some that stand out. Many of them live on my GitHub if you would like to go there to browse instead.

  • This website!

    Made completely from scratch using only HTML/CSS. It was created specifically to host my e-portfolio and see if I can still remember how to create a website after all of these years.

  • ELOsu

    An attempt at making unofficial ratings for osu! matches using khazhyk's osuapi package.

  • CS223 Coursework

    Different assignments and projects that I completed for my Scientific Programming in C (CS223) class.

  • "Top Text, Bottom Text" Image Maker

    A test of Python3 image processing using Pillow that I created based on the style of many old internet memes for a given input image and text.

  • Simple Discord Bot

    An IRC bot implementation for the IM service Discord using the discord.py wrapper in Python3. I created a few commands that show the variety of functions that can created using this setup.
