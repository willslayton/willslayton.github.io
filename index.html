Capstone Design Project: Note Taking App

For my senior capstone design project I worked with a team of three others, creating a speech-to-text application which uses a NVIDIA NeMo model to help create structured Markdown notes for users in the education domain. The machine learning aspect not only utilizes tailored speech recognition to more accurately match intended input, but can also use recognized command words to prompt certain interaction with the application. The simple GUI and text area, which can render many commonly used markdown symbols live, make operating the application easy and fast. Repository is private, but I may be able to provide further details upon request.

WX Language

WX is a personal project to try and create a programming language via a lexer, parser, and compiler in C. After taking a course on the organization of programming languages and learning about how languages are built I felt inspired to try and make something simple myself. Given the syntax described in my example the lexer tokenizes the text, which is then fed to a parser which generates an abstract syntax tree, and then will be compiled. I have gotten as far as adding some basic logic for the compilation translation layer to x32 and x64 assembly, but not much progress beyond since this is made in my free time.
